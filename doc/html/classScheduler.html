Detailed Description

Basic Scheduler.

This class represent a simple FIFO-Scheduler. It holds a list of the currently available threads. Through the Scheduler::resume method it can issue a context switch to another ready thread. The resulting context switch may also switch to the same thread if the ready list is empty.
