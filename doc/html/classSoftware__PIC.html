Detailed Description

Emulation of a Driver for the PIC (Programmable Interrupt Controller)

The PIC or Programmable Interrupt Controller enables the system to allow/forbid interrupts. By allowing an interrupt the PIC defines which interrupts are handled by the system. It is important that interrupts will only be handled if the interrupt has been allowed and if the handling of interrupts is enabled (see class Virtual_CPU).
