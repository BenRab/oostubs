"Objekt-Orientiertes Studenten BetriebSystem" (OOStuBS)

information

This framework helps student of computer of science to implement the practical tasks of the lecture TI-II. It provides abstraction and preimplemented components to ease the development of the needed functionality.

tasks

Task 1 - Input and Output
Task 2 - Interrupts
Task 3 - Analysation of the Procedurestack
Task 4 - Threads
Task 5 - Blocking Wait
Task 6 - Network
