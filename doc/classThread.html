Detailed Description

Basic multitasking entity.

Threads are the entities of scheduling. They are either part of the queue of threads ready for execution, blocked by a specific semaphore or currently running. This class defines the needed data structure to represent the context of a single thread(thread specific stack and thread control block). Additionally a C function is defined to help initiating the thread specific action.
